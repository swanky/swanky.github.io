講師介紹

本名蕭宇程(Hsiao Yu-Cheng)，巨蟹座Ａ型，國立臺灣師範大學資訊工程博士。 擅長 Java 程式設計與企業級服務架構、物件導向程式設計與架構、敏捷方法與敏捷軟體開發、 資料科學應用規畫、成長駭客、區塊鏈技術應用規畫及研發。曾執行大型雲端物聯聯網系統、 工業物聯網資料科學系統、邊緣資料中心基礎設施提案相關計畫。 現職東吳大學巨量資料管理學院區塊鏈課程兼任助理教授，曾任職國際信任機器技術總監。

課程內容

1. 加密貨幣簡介：加密貨幣、比特幣、區塊鏈、錢包、挖礦等等專有名詞的快速入門懶人包
2. 加密貨幣買賣：手把手教你如何註冊買幣平台與交易所，讓你可以買賣加密貨幣以及利用加密貨幣再進行投資
3. 相關資訊來源：介紹可以從哪裡獲得第一手新聞與知識，知己知彼投資更有利
