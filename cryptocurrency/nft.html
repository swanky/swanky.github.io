講師介紹

本名蕭宇程(Hsiao Yu-Cheng)，巨蟹座Ａ型，國立臺灣師範大學資訊工程博士。 擅長 Java 程式設計與企業級服務架構、物件導向程式設計與架構、敏捷方法與敏捷軟體開發、 資料科學應用規畫、成長駭客、區塊鏈技術應用規畫及研發。曾執行大型雲端物聯聯網系統、 工業物聯網資料科學系統、邊緣資料中心基礎設施提案相關計畫。 現職東吳大學巨量資料管理學院區塊鏈課程兼任助理教授，曾任職國際信任機器技術總監。

課程內容

最近元宇宙概念正夯，而加密貨幣與 NFT 會在元宇宙中扮演著極其重要的角色。 本課程帶您入門 NFT（非同質化代幣）並介紹其在各式應用中的生態系，無論您想成為 NFT 投資者或創作者都能夠馬上上手。 除了社群、藝術、影像、音樂之外，遊戲絕對是下一個能讓 NFT 展現其創新的領域，GameFi (Game Finance) / P2E (Play To Earn) 是什麼意思？區塊鏈遊戲是如何讓疫情中的東南亞國家瘋狂邊玩邊賺的。

1. 教你線上上架、買賣 NFT 一手包。利用話題性數字藝術品，開闢全新生財之道。
2. 介紹內行資訊蒐集，讓你在虛擬交易市場，如何無往不利。
3. 區塊鏈遊戲與現有的中心化遊戲之異同，GameFi 該如何邊玩邊賺。

安全買賣平台入門

網路上交易平台這麼多，哪個最方便好用同時也比較安全呢？ 本課程教你在台灣如何透過代買代賣平台購買比特幣，並透過交易所買賣其他各式競爭幣。 並介紹私人錢包讓你安全儲存自己的虛擬資產，不怕被駭客盜取。

技術資訊管道介紹

比特幣最近這麼熱門，但正反意見這麼多，有人說他會改變世界、也有人說他是泡沫是詐騙，到底真相為何？ 本課程以簡單易懂的介紹帶你快速瞭解比特幣及其他加密貨幣背後的區塊鍊技術，也會介紹相關看盤與新聞資訊來源，讓你透析局勢取得投資先機。

實務操作經驗分享

價格已經漲得這麼高了，現在買還來得及嗎？該如何才能賺到懷疑人生？ 本課程以系統實務操作的角度，手把手帶你一步一步實際進行演練。
